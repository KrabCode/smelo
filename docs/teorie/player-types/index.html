Typy hráčů

Každý hráč se dá zařadit podle dvou os: jak často hraje ruky (tight vs. loose) a jak agresivně je hraje (passive vs. aggressive).

Tight + Passive Nit
Tight + Aggressive TAG
Loose + Passive Calling Station
Loose + Aggressive LAG / Maniak
VPIP = % ruk, které hráč hraje (voluntary put money in pot)
PFR = % ruk, se kterými raisuje preflop
AF = aggression factor (bety + raisy / cally)

Vyber typ hráče

Klikni na kartu a procvič si situace proti danému typu.

TIGHT PASSIVE

Nit

Hraje málo ruk, spíš calluje. Raise = monster.

  • Folduje 80%+ ruk preflop
  • Raise na riveru = vždycky nuts
  • Málokdy 3-betuje
Exploit: Stealuj blindy, c-betuj, folduj na jeho agresi.
LOOSE PASSIVE

Calling Station

Calluje všechno. Nefolduje, nebluffuje.

  • Hraje 40%+ ruk preflop
  • Calluje s druhým párem na riveru
  • Pokud raisne — má to
Exploit: Value betuj i když máš hovno. Nebluffuj.
TIGHT AGGRESSIVE

TAG

Solidní styl. Málo ruk, ale hraje je agresivně.

  • 15–25% ruk preflop
  • C-betuje konzistentně
  • Dokáže foldovat
Exploit: 3-betuj light, floatuj c-bety, hledej leaky.
LOOSE AGGRESSIVE

LAG / Maniak

Hraje hodně, tlačí v každém potu. Nebezpečný.

  • 30%+ ruk preflop
  • Často 3-betuje a squeezuje
  • Bluffuje na všech streetech
Exploit: Rozšiř calling range, trapuj s premium.
Závěr: Nejde jen o to, jakou máš ruku. Jde o to, proti komu ji hraješ.
