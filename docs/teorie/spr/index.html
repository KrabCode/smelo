Stack-to-Pot Ratio

SPR je poměr efektivního stacku k potu na flopu. Určuje, jak moc jsi commitnutý a jak bys měl hrát postflop.

SPR = efektivní stack / pot na flopu

Efektivní stack = menší ze dvou stacků. Tolik můžeš maximálně vyhrát nebo prohrát.

SPR Zóny

SPR Zóna Důsledek
0–4 Nízký Commituj s top párem+ a silnými drawy. Binární rozhodování: all-in nebo fold.
4–10 Střední Na stack off potřebuješ dva páry+. Víc streetů, pozice je důležitější.
10+ Vysoký Deep stacked play. Spekulativní ruky (suited connectors, malé páry) nabírají hodnotu. Set mining je výdělečný.

Příklady

Tvůj stack: 500
Stack soupeře: 800
Efektivní stack: 500
Pot na flopu: 100

SPR: 500 / 100 = 5.0

Střední SPR. Top pár je dobrý, ale commitovat celý stack bez vylepšení je riskantní.
Tvůj stack: 200
Stack soupeře: 350
Efektivní stack: 200
Pot na flopu: 80

SPR: 200 / 80 = 2.5

Nízký SPR. Top pár s dobrým kickerem — dej all-in.

Preflop Sizing a SPR

Velikost preflop raisu přímo ovlivňuje SPR na flopu. Větší open = menší SPR = jednodušší postflop rozhodování.

S premium rukami (AA, KK) chceš nízký SPR — raisuj víc preflop. Se spekulativními rukami (suited connectors, malé páry) chceš vysoký SPR — drž pot malý a stacky hluboké.
